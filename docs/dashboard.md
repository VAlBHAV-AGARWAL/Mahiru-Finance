# Dashboard & Net Worth

## Pages

### Home Dashboard (`/dashboard`)
- Net worth card (Assets − Liabilities)
- Monthly income vs expense bar chart
- Spending by category (pie/donut chart)
- Recent transactions list (last 10)
- Upcoming bills / EMI alerts
- Budget progress bars (at-risk budgets)
- Quick-add transaction button

### Account Detail (`/accounts/[id]`)
- Balance history line chart
- Transactions filterable by date / category / tag
- Statement reconciliation status
- Edit / archive account

### All Transactions (`/transactions`)
- Table with filters: date range, account, category, tag, type, amount
- Search by description
- Bulk edit categories/tags
- Export to CSV

## India-specific Dashboard Metrics

- **Tax saved this year** (80C, NPS, etc.)
- **Investment allocation** (PPF vs MF vs FD vs Gold)
- **EMI burden** % of monthly income
- **UPI spend** vs total digital spend

## Charts (Recharts)

- Area: net worth over time
- Bar: monthly income vs expense
- Pie: category breakdown
- Line: individual account balance history
- Radar: budget adherence score
