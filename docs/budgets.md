# Budgets

## Model

| Field | Description |
|-------|-------------|
| `budgets` | Named budget plan (e.g. "Monthly Household") |
| `budget_limits` | Per-category limit with period (weekly/monthly/quarterly/yearly) |
| `budget_journal` | Links transactions to budgets for tracking |

## Features

1. **Create budget** — name, currency, active/inactive
2. **Set limits** — pick a category, set amount, choose period
3. **Track progress** — dashboard shows spent vs limit with % bar
4. **Warnings** — notifications when spending exceeds 80%, 90%, 100%
5. **Rollover** — optionally carry unused amount to next period

## India Context

- **Festival budget** — Diwali, wedding season (one-time quarterly)
- **Travel budget** — separate from monthly household
- **Education budget** — school fees, tuition
- **Medical budget** — insurance premiums + out-of-pocket
